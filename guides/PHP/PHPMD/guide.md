
# PHPMD

Wait for [php 8.0 support](https://github.com/phpmd/phpmd/issues/853).

## Install

```console
mkdir -p ./tools/phpmd
composer require --working-dir=./tools/phpmd phpmd/phpmd
```

## Update

```console
composer update --working-dir=./tools/phpmd
```

## Execute

```console
./tools/phpmd/vendor/bin/phpmd
```
